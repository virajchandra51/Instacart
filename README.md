# Instacart
Clone to the Original Website : instacart.com 

1.  Install dependencies:

        yarn

    or

        npm install

2. Install React:

        npm install react

3.  Run React:

        npm start

Your canvas piece should open up automatically at http://localhost:3000 

Tech Stack Used :

1. HTML5
2. CSS
3. Vanilla JavaScript
4. ViteJs
